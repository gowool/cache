# Cache

## Installation

### Install Cache

```sh
go get -u github.com/gowool/cache
```

### Install Fx Module

```sh
go get -u github.com/gowool/cache/fx
```

### Install Redis Backend

```sh
go get -u github.com/gowool/rediscachebe
```
